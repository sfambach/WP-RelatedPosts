# RelatedPosts

A native, modern Gutenberg block for WordPress 7+ that automatically displays a list of related posts based on automatic or manual categories, sorted descending by their last modification date.

This plugin was specifically designed and developed for [fambach.net](https://fambach.net).

## Features
- **WordPress 7+ Ready:** Full native support for modern PHP-only block registration.
- **Flexible Modes:** Use automatic category detection from the current post or manually overwrite it with comma-separated category IDs via the editor sidebar.
- **Subcategories:** Automatically includes child categories in the query.
- **Smart Sorting:** Sorts all related articles in descending order based on their actual last modification date.

## Installation
1. Download the latest `.zip` file from the **Releases** section on GitHub.
2. Log into your WordPress Dashboard and navigate to `Plugins > Add New > Upload Plugin`.
3. Choose the downloaded ZIP file and click `Install Now`.
4. Activate the plugin.
